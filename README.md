# Restaurant Menus
* An android sample example with MVI that helps the user to show the menu tags of the restaurant and each items
under this tag.
 
* requires Android Studio 3.4+ with android plugin for gradle 3.4+

## Dependencies
* Android X
* Retrofit
* RxJava
* RxAndroid
* ViewModel
* LiveData
* Room
* Dagger
* Kotlin
* Mockito
